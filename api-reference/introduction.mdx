<Note>

When using the API, make sure to replace any base URLs with `https://api.bieapi.com`. For example, if you see a URL like `https://api.openai.com`, you should use `https://api.bieapi.com` instead.

</Note>

## Authentication

All API endpoints are authenticated using Bearer tokens and picked up from the specification file.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
